## 3. Vertex AI 이미지 분류 (선택, 후순위)

- **목적**: 챌린지 인증샷(사진) 제출 시 텀블러/일회용컵 등 자동 분류.
- **필요**: GCP 프로젝트, Vertex AI API, 이미지 분류 모델(또는 커스텀 학습).
- **권장**: Phase 3에서 “설계만” 반영하고, 실제 연동은 별도 스프린트로 진행. 필요 시 `docs/plan-vertex-ai.md` 추가.

---

## 4. 모바일 웹뷰 UX 최적화

- **대상**: 메인, 기부, 이벤트, 마이페이지.
- **적용 사항**
  - 버튼·탭 최소 터치 영역 44x44px (Tailwind `min-h-[44px] min-w-[44px]` 등).
  - 테이블: 가로 스크롤 `overflow-x-auto` 또는 카드형 대체로 오버플로 방지.
  - 폰트: 본문 16px 이상 유지(줌 방지).
  - 하단 Navigation 터치 영역 충분히 확보 (이미 `py-2` 등 적용된 부분 점검).
